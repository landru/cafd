Computer-Aided Fish Design (cafd)

this is a set of python scripts that can be used to display wireframe models.
it depends on Pygame. The most up-to-date version, as of 12/24/2014, is in
the bitbucket.org account named polycopter.

cafd.py is the main script
simple3d.py implements the model class
view3d.py implements the view class

several objects are in the objects folder (a.k.a. directory)
